# Pokémon Compétitif — Encyclopédie Stratégique

Guide interactif complet pour maîtriser le Pokémon compétitif (Gen 1-9, Singles & Doubles/VGC).

## Fonctionnalités

- **15 sections** couvrant types, stats, talents, objets, statuts, attaques, mécaniques, teambuilding et plus
- **Pokédex complet** connecté à PokéAPI (1025 Pokémon) avec analyse compétitive et matchups
- **Base de données Moves** (900+ attaques) avec détails et effets secondaires
- **Calculateur de dégâts** interactif avec sélection de Pokémon via API
- **Team Builder** avec analyse en temps réel (faiblesses, couverture, rôles)
- **Quiz** (74+ questions) avec catégories et explications
- **Tooltips jargon** — survoler les termes compétitifs pour voir la définition
- **Thème clair/sombre** + taille de texte réglable
- **Responsive mobile**

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- PokéAPI (données en temps réel)

## Développement local

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Déploiement

Déployé automatiquement sur GitHub Pages via GitHub Actions à chaque push sur `main`.
